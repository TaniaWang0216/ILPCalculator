# 投資型保單配息計算機

這是一個可以直接在家用電腦使用的網頁版計算機。

## 使用方式

最簡單的方式：直接雙擊 `index.html`。

如果想讓同一台電腦用網址開啟：

```powershell
powershell -ExecutionPolicy Bypass -File .\start-server.ps1
```

然後打開：

```text
http://localhost:8080
```

## 目前公式

保費會先扣除級距手續費：

```text
保費大於或等於 2,000,000：手續費 3%
保費小於 2,000,000：手續費 3.2%
```

以 1,000,000 保費為例：

```text
實際投入金額 = 保費 * 0.968
```

固定配息：

```text
實際投入金額 / 標的淨值 * 0.04167
```

加發利息：

```text
實際投入金額 / 標的淨值 * 0.1
```

固定配息率 `0.04167` 與加發利息率 `0.1` 為固定值，網頁中不可更改。

金額採用小數捨去，所以預設範例會得到：

```text
月配息固定配息：3,798
月配息加發利息：9,114
每月合計：12,912
```
